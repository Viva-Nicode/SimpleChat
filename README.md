# 📬 SimpleChat

심플하고 가벼운 **채팅 애플리케이션**입니다.
필요한 메시지 기능만 담아, 빠르고 직관적으로 사용할 수 있도록 설계되었습니다.

## 🛠 프레임워크

- Swift
- Swiftui
- Combine / Swift Concurrency
- Alamofire
- Local Data (UserDefaults / CoreData / 기타)
- WebRTC
- SDWebImage
- XCTest

---

## 📱 소개

SimpleChat은 직관적인 UI와 가벼운 구조를 목표로 합니다.
불필요한 기능 없이 **채팅 경험에 집중**할 수 있도록 구성되어 있으며,
빠른 메시지 송수신과 안정적인 사용자 경험을 제공합니다.
